Nombre del proyecto 
Juego " Adivina el número" 

Integrante
Anthony Conlago

Objetivo del sistema
El objetivo del sistema que a travez de calculos matemáticos la computadora adivine el número que el usuario tenia en mente.

Descripción de funcionalidades
-Permite al usuario pensar un número dentro de un rango determinado.
-El computador realiza una primera suposición del número.
-El usuario indica si el número propuesto es mayor, menor o correcto.
-El algoritmo ajusta automáticamente el rango de búsqueda utilizando un proceso matemático (búsqueda binaria).
-El computador continúa realizando nuevas suposiciones hasta encontrar el número correcto.
-Muestra el número de intentos utilizados para adivinar el número.
-Finaliza el juego cuando el computador acierta el número pensado por el usuario.
